\section{Why standalone LaTeX?}

Unlike a \texttt{.qmd} file, a \texttt{.tex} file is compiled directly by a
LaTeX engine (\texttt{pdflatex} or \texttt{xelatex}) with no Pandoc step.
Positron has no built-in LaTeX compile button, so this file is compiled
from the terminal.

\section{Some math}

\begin{equation}

\section{Try it}

In the Positron terminal, run:
\begin{verbatim}
xelatex demo.tex
\end{verbatim}

Then open \texttt{demo.pdf} in Positron's PDF/file preview, or reveal it in
Finder.
